To get access to all of the coding examples in the React section, you have two options:

  1. View the code on Github here - no signup required
  2.  View the code on Cloud9 here

NOTE: You must have a cloud9 account to see the workspace, but don't worry, it's free.  

If you do not have access to a credit card please follow these instructions to get an account.  Remember, Cloud9 will not charge your credit card, it's purely for security reasons and to prevent bots from abusing their service.  I understand why they needed to implement this policy, but it's also very frustrating because I know many of students come from countries where credit cards are not commonly used.  Luckily, we have a workaround!

Send an email to webdevbootcampinfo@gmail.com and we can send you an invitation to c9 that does not require a credit card.  

Unfortunately, this is a very manual process for us, so it can take some time if we're asleep or away from email access. Please be patient with us!

If you're curious about why Cloud9 now requires a credit card to sign up, read a discussion on it here.Thanks for understanding!
