READ THIS BEFORE YOU START!

Before you move on to the React content in this course, make sure you are comfortable working with the following ES2015 and ES2016 concepts:

  • Promises
  • Const and Let
  • Arrow Functions
  • Spread Operator
  • Object Spread
  • Super keyword
  • Exports

React makes very very heavy use of the above. If you are not comfortable working with those new features, you will likely have a hard time learning React.  This is not your grandma's JavaScript! Things have changed, and modern JS looks very different than it used to.

Fortunately, we also have content to help catch you up on ES2015 and ES23016. We STRONGLY recommend you take an hour or two first to review our sections on ES2015 and ES2016. It won't take long!

If anyone actually reads these messages, thank you for making it this far! We wrote this because we noticed students were jumping straight to React without first going through the prerequisite content.  I know React is exciting, but trust us on this one, you don't want to try and learn things like arrow functions and the spread operator as you're also trying to learn React.
