Installing Node Without Cloud9 (not recommended) 

If you do not want to use Cloud9 to follow along with the Node parts of this course, you can instead install Node locally on your computer.  If you're set on installing locally, I still highly recommend you use Cloud9 because it's what I'll be using in the videos, and it's easier to get help when you have bugs.  If you're still set on installing locally, follow these instructions:

 Visit https://nodejs.org/en/download/ and download the version of Node that corresponds to your operating system and specific machine.  Once the download is complete, open up the package and run the installer.  

To verify the install, open up your command line and try running:

node -v 

If you see any version number (doesn't really matter which version), you're good to go!  If you get an error message instead, the installation may not have worked. Post on the discussion board to get some help.
