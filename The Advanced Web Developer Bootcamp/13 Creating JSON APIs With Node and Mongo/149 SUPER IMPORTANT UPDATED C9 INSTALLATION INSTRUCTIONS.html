Update as of December 29th 2017.

Hello Everyone! In the next lecture, I introduce c9, a cloud based development environment that you can use to develop your applications for the remainder of the course.  Cloud9 has changed the way they do signups since they were acquired by Amazon.  It still works as an installation option, but there are a couple of new steps.  If you want to use C9, you have a couple of options:

  • Follow these very detailed instructions on how to sign up for AWS C9 and install Node and Mongo.
  • Attempt this workaround using the old c9
  • As a last resort, send your email address to webdevbootcampinfo@gmail.com and we can send you an invitation to c9 that does not require a credit card or amazon web services signup. 

As always, you can install everything locally as well.
